## Decision Points Guide

### At Each "Conflicts?" Diamond:

**If YES:**
1. Click "Resolve conflicts" button
2. Edit each file manually
3. Remove conflict markers (`<<<<<<<`, `=======`, `>>>>>>>`)
4. Keep the code you want
5. Click "Mark as resolved"
6. Repeat for all files
7. Click "Commit merge"

**If NO:**
- Proceed directly to merge
- Click "Merge pull request"
- Click "Confirm merge"

### At "Tests Added?" Diamond:

**If YES:**
- Verify tests exist in PR files
- Check tests are passing
- Proceed to merge

**If NO:**
- Click "Review changes"
- Select "Request changes"
- Ask PR author to add tests
- OR add tests yourself via GitHub UI
- Wait for tests to pass
- Then proceed
